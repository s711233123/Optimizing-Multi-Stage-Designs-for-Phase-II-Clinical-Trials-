\section{研究背景}
\begin{frame}
\frametitle{研究背景}
\begin{itemize}
\item 說明多階段臨床試驗設計的為什麼重要，不僅是基於人道主義和成本把控、資源浪費、實驗效率、應對複雜問題。舉例Simon's兩階段設計如何幫助在提早終止無效試驗中，節省資源並減少病患風險（參考Simon 1989）。
\end{itemize}
\end{frame}

\begin{frame}
\frametitle{研究背景}
\begin{itemize}
\item 討論為何需要採用三階段設計來進一步最佳化樣本量，尤其是在藥物初步評估不佳的情況下，三階段設計能有效降低期望樣本量（參考Chen 1997）。

\item PSO如何實現多階段試驗設計的優勢。
\end{itemize}
\end{frame}
